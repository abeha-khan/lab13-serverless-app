# **App Name**: Function Runner

## Core Features:

- Ticket Price Calculation: Accept base price, demand, and days until the event; send to a serverless function, and display results. The response will include base price, demand, days until the event, final price, and price increase.
- Sum Calculation: Accept two numbers, send to a serverless function, and display the calculated sum.
- Loading State Handling: Show a loading indicator when data is being fetched from the serverless functions.
- Error Handling: Display user-friendly error messages if the serverless functions return an error.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey professionalism and trustworthiness.
- Background color: Light Gray (#F5F5F5), creating a clean and neutral backdrop.
- Accent color: Vibrant Cyan (#00BCD4) for interactive elements and call-to-action buttons.
- Body and headline font: 'PT Sans' (sans-serif) to offer a balance of modern aesthetic and readability.
- Use a clear, responsive layout with well-defined sections for each form.
- Use subtle animations (e.g., fading loading indicators) to provide feedback during asynchronous operations.
